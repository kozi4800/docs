## Bienvenue sur Red Frontier

Bienvenue dans l'univers de **Red Frontier**, un serveur RedM axé sur une immersion totale et une expérience de jeu enrichissante. Ce règlement est conçu pour garantir une expérience agréable, respectueuse et réaliste pour tous les joueurs.

### Objectif de cette documentation
Cette documentation a pour but de vous fournir toutes les informations nécessaires pour comprendre et respecter les règles de notre serveur, ainsi que d'optimiser votre expérience RolePlay.

---
